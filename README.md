# ETF Momentum Rankings

A web application to display ETF rising momentum rankings, designed for clarity and ease of update via daily captures.

## Features
- **Clean Dashboard**: "Ilmokyoyon" (well-organized) view of rankings.
- **Admin Tools**: Upload image -> OCR -> Update.
- **Responsive Design**: Works on mobile and desktop.
- **Korean Market Optimized**: Red for (+), Blue for (-).

## Tech Stack
- React + Vite
- TypeScript
- Tesseract.js (OCR)
- Lucide React (Icons)

## Setup
1. `npm install`
2. `npm run dev`

## Updating Data
Navigate to `/admin` to upload a new capture.
